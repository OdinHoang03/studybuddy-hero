# StudyBuddy — Landing Page

Cinematic single-page landing cho **StudyBuddy** (app tìm bạn học chung cho học sinh THPT). Làm cho **TheNextX Workshop 3 — Xây dựng Website cho Sản phẩm**.

Trang được dựng theo **5 section chuẩn** của một landing page: Hero · Problem · Solution · How it Works · CTA + Proof.

## Tech stack

- **React + Vite + TypeScript**
- **Tailwind CSS** + **shadcn/ui**
- **GSAP + ScrollTrigger** — reveal & parallax theo scroll
- **Lenis** — smooth scroll
- Typography editorial: **Instrument Serif** + **Inter**

## Chạy local

```bash
npm install
npm run dev      # http://localhost:5173
```

## Build

```bash
npm run build    # xuất ra dist/
npm run preview
```
